### Parity Calculation

```
   XOR Truth Table:
   0 XOR 0 = 0
   0 XOR 1 = 1
   1 XOR 0 = 1
   1 XOR 1 = 0

   Example:
   A1 = 1010, A2 = 1100, A3 = 0110
   Ap = 1010 XOR 1100 XOR 0110 = 0000

   If A2 lost:
   A2 = A1 XOR A3 XOR Ap
      = 1010 XOR 0110 XOR 0000
      = 1100 âœ“ Recovered!
```
