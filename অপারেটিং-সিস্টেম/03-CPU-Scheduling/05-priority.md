## ЁЯТ╗ ржЕрзНржпрж╛рж▓ржЧрж░рж┐ржжржо

```python
def priority_scheduling_non_preemptive(processes):
    current_time = 0
    completed = []
    
    while len(completed) < len(processes):
        # Available processes (arrived)
        available = [p for p in processes 
                     if p.arrival_time <= current_time 
                     and p not in completed]
        
        if available:
            # рж╕рж░рзНржмрзЛржЪрзНржЪ priority (рж╕ржмржЪрзЗржпрж╝рзЗ ржХржо рж╕ржВржЦрзНржпрж╛) ржмрж╛ржЫрж╛ржЗ
            available.sort(key=lambda x: x.priority)
            current_process = available[0]
            
            current_time += current_process.burst_time
            current_process.completion_time = current_time
            completed.append(current_process)
        else:
            current_time += 1
    
    return completed
```
