# ট্রান্সমিশন মিডিয়া (Transmission Media)

## 📖 ভূমিকা

**ট্রান্সমিশন মিডিয়া** হলো সেই মাধ্যম যার মধ্য দিয়ে ডেটা এক ডিভাইস থেকে অন্য ডিভাইসে যায়।

### প্রধান দুই ভাগ:
```
ট্রান্সমিশন মিডিয়া
       ├── তারযুক্ত (Wired/Guided)
       │      ├── টুইস্টেড পেয়ার
       │      ├── কোঅ্যাক্সিয়াল
       │      └── ফাইবার অপটিক
       │
       └── তারবিহীন (Wireless/Unguided)
              ├── রেডিও ওয়েভ
              ├── মাইক্রোওয়েভ
              └── ইনফ্রারেড
```

## 🔌 তারযুক্ত মিডিয়া (Guided Media)

---

### 🔗 ১. টুইস্টেড পেয়ার ক্যাবল (Twisted Pair Cable)

#### গঠন
দুটি তামার তার পরস্পর পেঁচিয়ে থাকে। এতে ইলেক্ট্রোম্যাগনেটিক ইন্টারফারেন্স কমে।

#### ডায়াগ্রাম
```
    ╭──────╮ ╭──────╮ ╭──────╮
───┤      ├─┤      ├─┤      ├───
    ╰──────╯ ╰──────╯ ╰──────╯
         টুইস্টেড পেয়ার
```

#### প্রকারভেদ

##### UTP (Unshielded Twisted Pair)
```
        [তার ১] ╭╮ [তার ২]
                ││
    কোনো শিল্ড নেই
```

| বৈশিষ্ট্য | বিবরণ |
|----------|--------|
| শিল্ড | নেই |
| দাম | সস্তা |
| ইনস্টল | সহজ |
| ইন্টারফারেন্স | বেশি |
| ব্যবহার | LAN, টেলিফোন |

##### STP (Shielded Twisted Pair)
```
        ┌──────────────┐
        │ [ফয়েল শিল্ড] │
        │  [তার ১╭╮তার ২] │
        └──────────────┘
```

| বৈশিষ্ট্য | বিবরণ |
|----------|--------|
| শিল্ড | আছে |
| দাম | বেশি |
| ইনস্টল | কঠিন |
| ইন্টারফারেন্স | কম |
| ব্যবহার | ইন্ডাস্ট্রিয়াল |

#### ক্যাটেগরি (Category)

| Cat | স্পিড | ব্যান্ডউইথ | ব্যবহার |
|-----|-------|-----------|---------|
| Cat 3 | 10 Mbps | 16 MHz | পুরাতন |
| Cat 5 | 100 Mbps | 100 MHz | পুরাতন |
| Cat 5e | 1 Gbps | 100 MHz | সাধারণ |
| **Cat 6** | 1 Gbps | 250 MHz | **জনপ্রিয়** |
| Cat 6a | 10 Gbps | 500 MHz | ডেটা সেন্টার |
| Cat 7 | 10 Gbps | 600 MHz | হাই পারফরম্যান্স |
| Cat 8 | 40 Gbps | 2000 MHz | ডেটা সেন্টার |

#### RJ-45 কানেক্টর
```
    ┌─────────────┐
    │ ┌─┬─┬─┬─┬─┬─┬─┬─┐ │
    │ │1│2│3│4│5│6│7│8│ │
    │ └─┴─┴─┴─┴─┴─┴─┴─┘ │
    └───────┬─────────┘
            │
        ক্যাবল
```

#### ক্যাবলিং স্ট্যান্ডার্ড

##### T568A
```
Pin 1: White/Green   Pin 5: Blue
Pin 2: Green         Pin 6: White/Orange
Pin 3: White/Orange  Pin 7: White/Brown
Pin 4: Blue          Pin 8: Brown
```

##### T568B (বেশি ব্যবহৃত)
```
Pin 1: White/Orange  Pin 5: Blue
Pin 2: Orange        Pin 6: White/Green
Pin 3: White/Green   Pin 7: White/Brown
Pin 4: Blue          Pin 8: Brown
```

#### ক্যাবলের ধরন

| ধরন | এক প্রান্ত | অন্য প্রান্ত | ব্যবহার |
|-----|-----------|-------------|---------|
| Straight-through | T568B | T568B | PC to Switch |
| Crossover | T568A | T568B | PC to PC |
| Rollover | Pin 1 | Pin 8 | Console |

### 📺 ২. কোঅ্যাক্সিয়াল ক্যাবল (Coaxial Cable)

#### গঠন
```
    ┌─────────────────────────────┐
    │ ═══════════════════════════ │ ← বাইরের জ্যাকেট
    │ ┌─────────────────────────┐ │
    │ │ ░░░░░░░░░░░░░░░░░░░░░░░ │ │ ← ব্রেইডেড শিল্ড
    │ │ ┌─────────────────────┐ │ │
    │ │ │                     │ │ │ ← ইনসুলেটর
    │ │ │ ═══════════════════ │ │ │ ← কেন্দ্রীয় কন্ডাক্টর
    │ │ └─────────────────────┘ │ │
    │ └─────────────────────────┘ │
    └─────────────────────────────┘
```

#### প্রকারভেদ

| প্রকার | RG নম্বর | Impedance | ব্যবহার |
|--------|----------|-----------|---------|
| Thicknet | RG-8 | 50Ω | পুরাতন Ethernet |
| Thinnet | RG-58 | 50Ω | পুরাতন Ethernet |
| Cable TV | RG-59 | 75Ω | টিভি, CCTV |
| Internet | RG-6 | 75Ω | কেবল ইন্টারনেট |

#### কানেক্টর

| কানেক্টর | ব্যবহার |
|----------|---------|
| BNC | পুরাতন নেটওয়ার্ক |
| F-Type | কেবল টিভি, ইন্টারনেট |

#### সুবিধা ও অসুবিধা

| সুবিধা ✅ | অসুবিধা ❌ |
|----------|----------|
| ভালো শিল্ডিং | ভারী ও শক্ত |
| দূরত্ব বেশি | ইনস্টল কঠিন |
| ইন্টারফারেন্স কম | দাম বেশি |

### 💡 ৩. ফাইবার অপটিক ক্যাবল (Fiber Optic Cable)

#### গঠন
```
    ┌─────────────────────────────────┐
    │         বাইরের জ্যাকেট           │
    │  ┌───────────────────────────┐  │
    │  │       বাফার কোটিং         │  │
    │  │  ┌─────────────────────┐  │  │
    │  │  │     ক্ল্যাডিং       │  │  │
    │  │  │  ┌───────────────┐  │  │  │
    │  │  │  │    কোর       │  │  │  │
    │  │  │  │ (আলো যায়)    │  │  │  │
    │  │  │  └───────────────┘  │  │  │
    │  │  └─────────────────────┘  │  │
    │  └───────────────────────────┘  │
    └─────────────────────────────────┘
```

#### কাজের নীতি
**Total Internal Reflection** - আলো কোরে প্রবেশ করে এবং ক্ল্যাডিং থেকে বাউন্স করে এগিয়ে যায়।

```
         ╱╲    ╱╲    ╱╲
   →────╱  ╲  ╱  ╲  ╱  ╲────→
   আলো   ╲  ╱╲  ╱╲  ╱   আলো
          ╲╱  ╲╱  ╲╱
```

#### প্রকারভেদ

##### Single-Mode Fiber (SMF)
```
    ─────────────────────
    │ ═════════════════ │ ← একক আলোক রশ্মি
    ─────────────────────
    কোর: ~9 µm
```

| বৈশিষ্ট্য | বিবরণ |
|----------|--------|
| কোর সাইজ | ৮-১০ µm |
| দূরত্ব | ১০০+ কিমি |
| স্পিড | অনেক বেশি |
| খরচ | বেশি |
| ব্যবহার | WAN, টেলিকম |

##### Multi-Mode Fiber (MMF)
```
    ─────────────────────
    │ ══╱╲══╲╱══╱╲══ │ ← একাধিক আলোক রশ্মি
    ─────────────────────
    কোর: ~50-62.5 µm
```

| বৈশিষ্ট্য | বিবরণ |
|----------|--------|
| কোর সাইজ | ৫০-৬২.৫ µm |
| দূরত্ব | ২ কিমি পর্যন্ত |
| স্পিড | বেশি |
| খরচ | কম |
| ব্যবহার | LAN, ডেটা সেন্টার |

#### কানেক্টর

| কানেক্টর | বিবরণ |
|----------|--------|
| SC | Square Connector, পুশ-পুল |
| LC | Lucent Connector, ছোট |
| ST | Straight Tip, বেয়নেট |
| FC | Ferrule Connector, স্ক্রু |
| MPO | Multi-fiber, ডেটা সেন্টার |

#### Copper vs Fiber

| বৈশিষ্ট্য | Copper | Fiber |
|----------|--------|-------|
| স্পিড | 10 Gbps পর্যন্ত | 100+ Gbps |
| দূরত্ব | ১০০ মিটার | ১০০+ কিমি |
| ইন্টারফারেন্স | আছে | নেই |
| সিকিউরিটি | ট্যাপ করা সহজ | কঠিন |
| খরচ | কম | বেশি |
| ইনস্টল | সহজ | কঠিন |
| ওজন | ভারী | হালকা |

## 📡 তারবিহীন মিডিয়া (Unguided Media)

---

### 📻 ১. রেডিও ওয়েভ (Radio Waves)

#### বৈশিষ্ট্য
- ফ্রিকোয়েন্সি: 3 kHz - 1 GHz
- সর্বমুখী (সব দিকে যায়)
- দেয়াল ভেদ করতে পারে

#### ব্যবহার
| প্রযুক্তি | ফ্রিকোয়েন্সি | পরিসর |
|----------|------------|-------|
| AM Radio | 535-1705 kHz | দূরপাল্লা |
| FM Radio | 88-108 MHz | মাঝারি |
| Wi-Fi | 2.4/5 GHz | ছোট |
| Bluetooth | 2.4 GHz | খুব ছোট |
| Cellular | 700-2600 MHz | বড় |

---

### 📶 ২. মাইক্রোওয়েভ (Microwave)

#### বৈশিষ্ট্য
- ফ্রিকোয়েন্সি: 1 GHz - 300 GHz
- দিকনির্দেশক (একদিকে যায়)
- Line-of-sight প্রয়োজন

#### প্রকারভেদ

##### Terrestrial Microwave
```
    📡 ─────────────────────── 📡
  Tower A      সরাসরি দৃষ্টি     Tower B
```

| বৈশিষ্ট্য | বিবরণ |
|----------|--------|
| দূরত্ব | ৫০-৮০ কিমি (টাওয়ার থেকে টাওয়ার) |
| ব্যবহার | টেলিফোন, ব্যাকবোন |
| সমস্যা | আবহাওয়া প্রভাবিত করে |

##### Satellite Microwave
```
        🛰️ স্যাটেলাইট
       ╱     ╲
      ╱       ╲
    📡         📡
  Station A   Station B
```

| বৈশিষ্ট্য | বিবরণ |
|----------|--------|
| দূরত্ব | বিশ্বব্যাপী |
| Latency | বেশি (৫০০+ ms) |
| ব্যবহার | টিভি, GPS, ইন্টারনেট |

---

### 🔴 ৩. ইনফ্রারেড (Infrared)

#### বৈশিষ্ট্য
- ফ্রিকোয়েন্সি: 300 GHz - 400 THz
- খুব ছোট পরিসর
- দেয়াল ভেদ করতে পারে না
- Line-of-sight প্রয়োজন

#### ব্যবহার
```
📱 ════════════════════ 📺
রিমোট      ইনফ্রারেড      টিভি
```

- TV রিমোট
- পুরাতন মোবাইল ট্রান্সফার
- IrDA (Infrared Data Association)
