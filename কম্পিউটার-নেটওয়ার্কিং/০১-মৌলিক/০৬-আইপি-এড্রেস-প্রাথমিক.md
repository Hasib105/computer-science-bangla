# IP Address বেসিক (IP Address Basics)

## 📖 IP Address কী?

**IP Address (Internet Protocol Address)** হলো নেটওয়ার্কে প্রতিটি ডিভাইসের একটি ইউনিক আইডেন্টিফায়ার বা ঠিকানা।

### উদাহরণ
```
আপনার বাসার ঠিকানা: বাড়ি ১২, রোড ৫, ধানমন্ডি
IP Address: 192.168.1.10
```

---

## 🔢 IP Address এর গঠন (IPv4)

### বেসিক ফরম্যাট
```
    192  .  168  .  1    .  10
     ↓       ↓      ↓       ↓
  Octet 1  Octet 2 Octet 3 Octet 4
```

### বৈশিষ্ট্য
- ৪টি অক্টেট (Octet)
- প্রতিটি অক্টেট ৮ বিট
- মোট ৩২ বিট
- ডট দিয়ে আলাদা করা

### বাইনারি রূপ
```
Decimal:  192     .  168     .  1       .  10
Binary:   11000000.10101000.00000001.00001010
```

### প্রতিটি অক্টেটের রেঞ্জ
```
সর্বনিম্ন: 0   (00000000)
সর্বোচ্চ:  255 (11111111)
```

---

## 🔄 ডেসিমাল থেকে বাইনারি রূপান্তর

### পদ্ধতি
প্রতিটি বিটের মান:
```
128  64  32  16  8  4  2  1
 ↓    ↓   ↓   ↓  ↓  ↓  ↓  ↓
2⁷  2⁶  2⁵  2⁴ 2³ 2² 2¹ 2⁰
```

### উদাহরণ: 192 কে বাইনারিতে
```
192 ÷ 128 = 1 (বাকি 64)    → 1
64  ÷ 64  = 1 (বাকি 0)     → 1
0   ÷ 32  = 0              → 0
0   ÷ 16  = 0              → 0
0   ÷ 8   = 0              → 0
0   ÷ 4   = 0              → 0
0   ÷ 2   = 0              → 0
0   ÷ 1   = 0              → 0

ফলাফল: 11000000
```

### শর্টকাট পদ্ধতি
```
128 + 64 = 192
 ↓    ↓
 1    1    0    0    0    0    0    0
```

## 📊 IP Address এর অংশ

প্রতিটি IP Address দুটি অংশে বিভক্ত:

### ১. Network ID
- নেটওয়ার্ক চিহ্নিত করে
- একই নেটওয়ার্কের সব ডিভাইসে একই

### ২. Host ID
- ডিভাইস চিহ্নিত করে
- প্রতিটি ডিভাইসে আলাদা

### উদাহরণ
```
IP: 192.168.1.10
    └──┬──┘ └┬┘
   Network  Host
     ID      ID
```

## 🏷️ IP Address এর ক্লাস

### ক্লাস টেবিল

| ক্লাস | প্রথম অক্টেট | ডিফল্ট সাবনেট মাস্ক | নেটওয়ার্ক | হোস্ট |
|-------|-------------|-------------------|----------|-------|
| A | 1-126 | 255.0.0.0 | বড় অর্গানাইজেশন | ~16 মিলিয়ন |
| B | 128-191 | 255.255.0.0 | মাঝারি অর্গানাইজেশন | ~65,000 |
| C | 192-223 | 255.255.255.0 | ছোট নেটওয়ার্ক | 254 |
| D | 224-239 | - | Multicast | - |
| E | 240-255 | - | Reserved | - |

### ভিজুয়াল রিপ্রেজেন্টেশন

#### Class A
```
N  .  H  .  H  .  H
│     └─────┬─────┘
│        Host ID
└── Network ID
```

#### Class B
```
N  .  N  .  H  .  H
└──┬──┘    └──┬──┘
Network    Host ID
  ID
```

#### Class C
```
N  .  N  .  N  .  H
└─────┬─────┘    │
   Network      Host
     ID          ID
```

### ক্লাস চিহ্নিতকরণ (প্রথম বিট)
```
Class A: 0xxxxxxx    (0-127)
Class B: 10xxxxxx    (128-191)
Class C: 110xxxxx    (192-223)
Class D: 1110xxxx    (224-239)
Class E: 1111xxxx    (240-255)
```

## 🏠 পাবলিক vs প্রাইভেট IP

### পাবলিক IP
- ইন্টারনেটে ইউনিক
- ISP দেয়
- ইন্টারনেট থেকে অ্যাক্সেস করা যায়

### প্রাইভেট IP
- লোকাল নেটওয়ার্কে ব্যবহৃত
- ইন্টারনেটে রাউটেবল না
- ফ্রি, বারবার ব্যবহার করা যায়

### প্রাইভেট IP রেঞ্জ

| ক্লাস | রেঞ্জ | সাবনেট | ব্যবহার |
|-------|-------|--------|---------|
| A | 10.0.0.0 - 10.255.255.255 | 10.0.0.0/8 | বড় অর্গানাইজেশন |
| B | 172.16.0.0 - 172.31.255.255 | 172.16.0.0/12 | মাঝারি |
| C | 192.168.0.0 - 192.168.255.255 | 192.168.0.0/16 | হোম/ছোট অফিস |

### বাসায় সাধারণত ব্যবহৃত
```
192.168.0.x
192.168.1.x
10.0.0.x
```

## 🎭 বিশেষ IP Addresses

| IP Address | ব্যবহার |
|------------|---------|
| 0.0.0.0 | Default Route, "এই নেটওয়ার্ক" |
| 127.0.0.1 | Localhost (নিজের কম্পিউটার) |
| 127.0.0.0 - 127.255.255.255 | Loopback Range |
| 255.255.255.255 | Broadcast (সবাইকে পাঠানো) |
| 169.254.0.0 - 169.254.255.255 | APIPA (Auto-assigned) |

### Localhost টেস্ট
```cmd
ping 127.0.0.1
```
এটা কাজ করলে TCP/IP ঠিকমতো ইনস্টল আছে।

## 🎨 সাবনেট মাস্ক (Subnet Mask)

### সংজ্ঞা
সাবনেট মাস্ক Network ID এবং Host ID আলাদা করে।

### কাজের পদ্ধতি
```
IP Address:    192.168.1.10      11000000.10101000.00000001.00001010
Subnet Mask:   255.255.255.0     11111111.11111111.11111111.00000000
               ───────────────────────────────────────────────────────
Network ID:    192.168.1.0       11000000.10101000.00000001.00000000
```

### সাধারণ সাবনেট মাস্ক

| ক্লাস | সাবনেট মাস্ক | CIDR | নেটওয়ার্ক বিট |
|-------|-------------|------|--------------|
| A | 255.0.0.0 | /8 | 8 |
| B | 255.255.0.0 | /16 | 16 |
| C | 255.255.255.0 | /24 | 24 |

### CIDR Notation
```
192.168.1.0/24
            ↓
         24টি বিট Network ID
```

## 📡 স্ট্যাটিক vs ডাইনামিক IP

### Static IP
- ম্যানুয়ালি সেট করা
- পরিবর্তন হয় না
- সার্ভার, প্রিন্টারের জন্য

```
IP: 192.168.1.100
Subnet: 255.255.255.0
Gateway: 192.168.1.1
DNS: 8.8.8.8
```

### Dynamic IP (DHCP)
- অটোমেটিক পাওয়া যায়
- DHCP সার্ভার দেয়
- সাধারণ ডিভাইসের জন্য

```
[PC] ──→ "IP দাও" ──→ [DHCP Server]
     ←── "এই নাও 192.168.1.50" ←──
```

## 🌐 IPv4 vs IPv6 (সংক্ষেপে)

| বৈশিষ্ট্য | IPv4 | IPv6 |
|----------|------|------|
| বিট সংখ্যা | 32 | 128 |
| ফরম্যাট | Decimal (.) | Hexadecimal (:) |
| উদাহরণ | 192.168.1.1 | 2001:0db8:85a3::8a2e |
| সংখ্যা | ~4.3 বিলিয়ন | ~340 undecillion |

IPv6 বিস্তারিত পরে আলোচনা করা হবে।
