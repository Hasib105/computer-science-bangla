# নেটওয়ার্কের প্রকারভেদ (Types of Networks)

## 📖 ভৌগোলিক বিস্তারের ভিত্তিতে নেটওয়ার্কের প্রকার

---

## 🏠 ১. PAN (Personal Area Network)

### সংজ্ঞা
ব্যক্তিগত ডিভাইসগুলোর মধ্যে ক্ষুদ্র পরিসরের নেটওয়ার্ক।

### বৈশিষ্ট্য
- পরিসর: ১০ মিটারের মধ্যে
- ব্যক্তিগত ব্যবহার
- কম খরচ

### উদাহরণ
```
📱 ফোন ←→ 🎧 ব্লুটুথ হেডফোন
💻 ল্যাপটপ ←→ ⌚ স্মার্টওয়াচ
🖥️ কম্পিউটার ←→ 🖨️ ওয়্যারলেস প্রিন্টার
```

### প্রযুক্তি
- Bluetooth
- USB
- Infrared (IR)
- NFC (Near Field Communication)

---

## 🏢 ২. LAN (Local Area Network)

### সংজ্ঞা
একটি বিল্ডিং বা ক্যাম্পাসের মধ্যে সীমাবদ্ধ নেটওয়ার্ক।

### বৈশিষ্ট্য
- পরিসর: ১ কিমি পর্যন্ত
- হাই স্পিড (100 Mbps - 10 Gbps)
- প্রাইভেট নেটওয়ার্ক
- কম খরচ
- সহজ মেইনটেন্যান্স

### উদাহরণ
```
🏫 স্কুলের কম্পিউটার ল্যাব
🏢 অফিসের নেটওয়ার্ক
🏠 হোম নেটওয়ার্ক
🏥 হাসপাতালের নেটওয়ার্ক
```

### ডায়াগ্রাম
```
        [সার্ভার]
            |
        [সুইচ]
       /   |   \
   [PC1] [PC2] [PC3]
```

### প্রযুক্তি
- Ethernet (IEEE 802.3)
- Wi-Fi (IEEE 802.11)
- Token Ring (পুরাতন)

---

## 🏙️ ৩. MAN (Metropolitan Area Network)

### সংজ্ঞা
একটি শহর বা মেট্রোপলিটন এরিয়া জুড়ে বিস্তৃত নেটওয়ার্ক।

### বৈশিষ্ট্য
- পরিসর: ৫০-১০০ কিমি
- একাধিক LAN যুক্ত করে
- মাঝারি থেকে উচ্চ খরচ
- ISP বা সরকার পরিচালনা করে

### উদাহরণ
```
🏛️ শহরের সব সরকারি অফিস একসাথে যুক্ত
📺 কেবল টিভি নেটওয়ার্ক
🏦 শহরের সব ব্যাংক শাখা যুক্ত
```

### ডায়াগ্রাম
```
[LAN-অফিস ১] ←→ [MAN ব্যাকবোন] ←→ [LAN-অফিস ২]
                    ↕
              [LAN-অফিস ৩]
```

### প্রযুক্তি
- Fiber Optic
- Metro Ethernet
- WiMAX

---

## 🌍 ৪. WAN (Wide Area Network)

### সংজ্ঞা
বিশাল ভৌগোলিক এলাকা জুড়ে বিস্তৃত নেটওয়ার্ক। দেশ বা মহাদেশ জুড়ে থাকতে পারে।

### বৈশিষ্ট্য
- পরিসর: দেশ/বিশ্বব্যাপী
- LAN এবং MAN গুলো যুক্ত করে
- উচ্চ খরচ
- তুলনামূলক কম স্পিড (LAN এর চেয়ে)
- পাবলিক বা প্রাইভেট হতে পারে

### উদাহরণ
```
🌐 ইন্টারনেট (সবচেয়ে বড় WAN)
🏦 মাল্টিন্যাশনাল ব্যাংকের নেটওয়ার্ক
🛒 Amazon, Google এর গ্লোবাল নেটওয়ার্ক
```

### ডায়াগ্রাম
```
   [ঢাকা অফিস]          [নিউইয়র্ক অফিস]
        |                      |
   [Router] ←-- ইন্টারনেট --→ [Router]
        |                      |
   [LAN]                   [LAN]
```

### প্রযুক্তি
- Leased Lines
- MPLS
- Satellite
- Frame Relay

---

## 🔄 ৫. অন্যান্য নেটওয়ার্ক প্রকার

### WLAN (Wireless LAN)
- তারবিহীন লোকাল এরিয়া নেটওয়ার্ক
- Wi-Fi ব্যবহার করে
- মোবাইল ডিভাইসের জন্য আদর্শ

### CAN (Campus Area Network)
- বিশ্ববিদ্যালয় ক্যাম্পাস বা কর্পোরেট ক্যাম্পাসে
- LAN এর চেয়ে বড়, MAN এর চেয়ে ছোট
- একাধিক বিল্ডিং কভার করে

### SAN (Storage Area Network)
- স্টোরেজ ডিভাইসের জন্য বিশেষ নেটওয়ার্ক
- ডেটা সেন্টারে ব্যবহৃত
- হাই-স্পিড ডেটা ট্রান্সফার

### VPN (Virtual Private Network)
- পাবলিক নেটওয়ার্কে প্রাইভেট কানেকশন
- এনক্রিপ্টেড
- রিমোট এক্সেস

---

## 📊 তুলনামূলক চার্ট

| বৈশিষ্ট্য | PAN | LAN | MAN | WAN |
|----------|-----|-----|-----|-----|
| পরিসর | ১০ মি | ১ কিমি | ১০০ কিমি | দেশ/বিশ্ব |
| স্পিড | মাঝারি | উচ্চ | মাঝারি | কম-মাঝারি |
| খরচ | কম | কম | মাঝারি | উচ্চ |
| মালিকানা | ব্যক্তিগত | প্রাইভেট | মিক্সড | মিক্সড |
| উদাহরণ | Bluetooth | অফিস | শহর | ইন্টারনেট |

---

## 🎨 ভিজুয়াল তুলনা

```
PAN (ব্যক্তি)     →     LAN (বিল্ডিং)     →     MAN (শহর)     →     WAN (দেশ/বিশ্ব)
   👤                     🏢                    🏙️                   🌍
 ~10m                   ~1km                 ~100km               unlimited
```

## 🔧 ব্যবহারিক পার্থক্য

### হোম নেটওয়ার্ক (LAN + PAN)
```
ইন্টারনেট (WAN)
      ↓
   [Router]
      ↓
[LAN - ইথারনেট/Wi-Fi]
   /    |    \
[PC]  [TV]  [Phone]
              ↓
        [PAN - Bluetooth]
              ↓
         [Earbuds]
```
