# নেটওয়ার্ক ডিভাইস (Network Devices)

## 📖 ভূমিকা

নেটওয়ার্ক ডিভাইস হলো সেই হার্ডওয়্যার যা কম্পিউটার এবং অন্যান্য ডিভাইসগুলোকে নেটওয়ার্কে সংযুক্ত করে এবং ডেটা আদান-প্রদানে সাহায্য করে।

---

## 🔌 ১. NIC (Network Interface Card)

### সংজ্ঞা
কম্পিউটারকে নেটওয়ার্কে যুক্ত করার জন্য ব্যবহৃত কার্ড।

### ডায়াগ্রাম
```
[কম্পিউটার]
     |
  [NIC] ←── ইথারনেট পোর্ট
     |
  [ক্যাবল]
```

### বৈশিষ্ট্য
- **MAC Address**: প্রতিটি NIC এর ইউনিক ফিজিক্যাল অ্যাড্রেস
- **স্পিড**: 10/100/1000 Mbps (Gigabit)
- **ধরন**: Wired (RJ-45) বা Wireless (Wi-Fi)

### MAC Address উদাহরণ
```
00:1A:2B:3C:4D:5E
    ↓
6 জোড়া হেক্সাডেসিমাল সংখ্যা
```

### NIC এর কাজ
1. ডেটাকে ইলেকট্রিক্যাল সিগন্যালে রূপান্তর
2. ডেটা পাঠানো এবং গ্রহণ
3. এরর চেকিং
4. MAC অ্যাড্রেসিং

---

## 🔁 ২. রিপিটার (Repeater)

### সংজ্ঞা
দুর্বল সিগন্যালকে শক্তিশালী করে পুনরায় পাঠায়।

### ডায়াগ্রাম
```
[PC1] ----দুর্বল সিগন্যাল---→ [Repeater] ----শক্তিশালী সিগন্যাল---→ [PC2]
```

### কাজ
- সিগন্যাল Amplify করে
- দূরত্ব বাড়ায়
- OSI Layer 1 (Physical Layer) এ কাজ করে

### সীমাবদ্ধতা
- শুধু সিগন্যাল বাড়ায়, ফিল্টার করে না
- নেটওয়ার্ক ট্রাফিক কমায় না

---

## 📡 ৩. হাব (Hub)

### সংজ্ঞা
একাধিক ডিভাইসকে একসাথে যুক্ত করে। এটি একটি মাল্টি-পোর্ট রিপিটার।

### ডায়াগ্রাম
```
          [PC1]
            |
    [PC4]---[HUB]---[PC2]
            |
          [PC3]
```

### প্রকারভেদ

| প্রকার | বিবরণ |
|--------|--------|
| Passive Hub | শুধু কানেক্ট করে, সিগন্যাল বাড়ায় না |
| Active Hub | সিগন্যাল Regenerate করে |
| Intelligent Hub | ম্যানেজমেন্ট ফিচার আছে |

### কাজের পদ্ধতি
```
PC1 ডেটা পাঠায় → Hub সবাইকে পাঠায় → সবাই পায় → যার জন্য, সে রাখে
```

### সমস্যা ⚠️
- **Broadcast**: সব পোর্টে ডেটা যায়
- **Collision**: একসাথে ডেটা পাঠালে সংঘর্ষ
- **সিকিউরিটি**: সবাই সব ডেটা দেখতে পায়
- **Half Duplex**: একসাথে পাঠানো ও গ্রহণ করা যায় না

### বর্তমান অবস্থা
❌ **Hub এখন আর ব্যবহৃত হয় না। Switch ব্যবহার করা হয়।**

---

## 🔀 ৪. সুইচ (Switch)

### সংজ্ঞা
বুদ্ধিমান ডিভাইস যা MAC Address দেখে সঠিক পোর্টে ডেটা পাঠায়।

### ডায়াগ্রাম
```
          [PC1]
            |
    [PC4]---[Switch]---[PC2]
            |
          [PC3]
```

### Hub vs Switch

| বৈশিষ্ট্য | Hub | Switch |
|----------|-----|--------|
| ডেটা পাঠানো | সব পোর্টে | নির্দিষ্ট পোর্টে |
| Collision | আছে | নেই |
| স্পিড | ধীর | দ্রুত |
| সিকিউরিটি | কম | বেশি |
| Duplex | Half | Full |
| দাম | সস্তা | বেশি |

### কাজের পদ্ধতি
```
PC1 (MAC: AA) → Switch → MAC Table দেখে → PC2 (MAC: BB) এর পোর্টে পাঠায়
```

### MAC Address Table
```
| Port | MAC Address      |
|------|------------------|
| 1    | 00:1A:2B:3C:4D:5E|
| 2    | 00:1A:2B:3C:4D:5F|
| 3    | 00:1A:2B:3C:4D:60|
```

### প্রকারভেদ

| প্রকার | বিবরণ |
|--------|--------|
| Unmanaged | সাধারণ, কনফিগার করা যায় না |
| Managed | কনফিগারযোগ্য, VLAN সাপোর্ট |
| Layer 3 Switch | রাউটিং করতে পারে |
| PoE Switch | Power over Ethernet সাপোর্ট |

---

## 🌐 ৫. রাউটার (Router)

### সংজ্ঞা
বিভিন্ন নেটওয়ার্ককে সংযুক্ত করে এবং IP Address দেখে সেরা পথে ডেটা পাঠায়।

### ডায়াগ্রাম
```
[Network A: 192.168.1.0]          [Network B: 192.168.2.0]
        |                                  |
      [PC1]                              [PC2]
        |                                  |
   [Switch A]                         [Switch B]
        |                                  |
        └────────[Router]──────────────────┘
```

### প্রধান কাজ
1. **রাউটিং**: সেরা পথ নির্বাচন
2. **নেটওয়ার্ক সংযোগ**: বিভিন্ন নেটওয়ার্ক যুক্ত করা
3. **NAT**: প্রাইভেট IP কে পাবলিক IP তে রূপান্তর
4. **DHCP**: অটোমেটিক IP দেওয়া
5. **ফায়ারওয়াল**: বেসিক সিকিউরিটি

### Routing Table উদাহরণ
```
| Destination    | Gateway       | Interface |
|----------------|---------------|-----------|
| 192.168.1.0/24 | 0.0.0.0       | eth0      |
| 192.168.2.0/24 | 0.0.0.0       | eth1      |
| 0.0.0.0/0      | 203.0.113.1   | eth2      |
```

### Switch vs Router

| বৈশিষ্ট্য | Switch | Router |
|----------|--------|--------|
| Layer | Layer 2 | Layer 3 |
| Address | MAC | IP |
| Broadcast | পাঠায় | ব্লক করে |
| নেটওয়ার্ক | একই নেটওয়ার্ক | বিভিন্ন নেটওয়ার্ক |
| দাম | কম | বেশি |

---

## 🌉 ৬. ব্রিজ (Bridge)

### সংজ্ঞা
দুটি নেটওয়ার্ক সেগমেন্টকে যুক্ত করে।

### ডায়াগ্রাম
```
[Segment 1]          [Segment 2]
  [PC1]                [PC3]
  [PC2]                [PC4]
    |                    |
    └───[Bridge]─────────┘
```

### কাজ
- MAC Address দেখে ডেটা ফিল্টার করে
- Collision Domain আলাদা করে
- Switch এর পূর্বসূরি

---

## 📶 ৭. মডেম (Modem)

### সংজ্ঞা
**Mo**dulator + **Dem**odulator = Modem

ডিজিটাল সিগন্যালকে এনালগে এবং এনালগকে ডিজিটালে রূপান্তর করে।

### ডায়াগ্রাম
```
[কম্পিউটার] ←ডিজিটাল→ [Modem] ←এনালগ→ [টেলিফোন লাইন] ←→ [ISP]
```

### প্রকারভেদ

| প্রকার | বিবরণ | স্পিড |
|--------|--------|-------|
| Dial-up | পুরাতন, ফোন লাইন ব্যবহার | 56 Kbps |
| DSL | ফোন লাইনে হাই স্পিড | 1-100 Mbps |
| Cable | কেবল TV লাইন | 10-500 Mbps |
| Fiber | অপটিক্যাল ফাইবার | 100+ Mbps |
| Cellular | মোবাইল নেটওয়ার্ক (3G/4G/5G) | Variable |

## 🔒 ৮. গেটওয়ে (Gateway)

### সংজ্ঞা
বিভিন্ন প্রোটোকল ব্যবহারকারী নেটওয়ার্ককে সংযুক্ত করে।

### ডায়াগ্রাম
```
[TCP/IP Network]          [Other Protocol Network]
       |                           |
       └─────[Gateway]─────────────┘
              (প্রোটোকল রূপান্তর)
```

### উদাহরণ
- Email Gateway
- VoIP Gateway
- Cloud Gateway

## 📡 ৯. অ্যাক্সেস পয়েন্ট (Access Point)

### সংজ্ঞা
ওয়্যারলেস ডিভাইসগুলোকে ওয়্যার্ড নেটওয়ার্কে যুক্ত করে।

### ডায়াগ্রাম
```
[Router]───[Switch]───[Access Point]))
                            |
                      📶 Wi-Fi Signal
                      /    |    \
                  📱    💻    📱
```

### Wireless Router vs Access Point

| বৈশিষ্ট্য | Wireless Router | Access Point |
|----------|-----------------|--------------|
| রাউটিং | করতে পারে | পারে না |
| DHCP | আছে | নেই |
| NAT | আছে | নেই |
| ব্যবহার | হোম | এন্টারপ্রাইজ |

## 🛡️ ১০. ফায়ারওয়াল (Firewall)

### সংজ্ঞা
নেটওয়ার্ক সিকিউরিটি ডিভাইস যা ইনকামিং এবং আউটগোয়িং ট্রাফিক নিয়ন্ত্রণ করে।

### ডায়াগ্রাম
```
[Internet] ←→ [Firewall] ←→ [Internal Network]
                  ↓
           ✓ Allow/Block
```

## 📊 OSI Layer অনুযায়ী ডিভাইস

```
Layer 7 (Application)  → Gateway, Firewall
Layer 6 (Presentation) → Gateway
Layer 5 (Session)      → Gateway
Layer 4 (Transport)    → Firewall
Layer 3 (Network)      → Router, Layer 3 Switch
Layer 2 (Data Link)    → Switch, Bridge
Layer 1 (Physical)     → Hub, Repeater, NIC, Modem
```

---

## 📋 সারসংক্ষেপ

| ডিভাইস | Layer | কাজ |
|--------|-------|-----|
| NIC | 1,2 | কম্পিউটারকে নেটওয়ার্কে যুক্ত করে |
| Repeater | 1 | সিগন্যাল বাড়ায় |
| Hub | 1 | সব পোর্টে ডেটা পাঠায় |
| Switch | 2 | MAC দেখে সঠিক পোর্টে পাঠায় |
| Bridge | 2 | দুটি সেগমেন্ট যুক্ত করে |
| Router | 3 | IP দেখে পথ নির্বাচন করে |
| Gateway | 7 | প্রোটোকল রূপান্তর করে |
| Firewall | 3,4,7 | সিকিউরিটি দেয় |
