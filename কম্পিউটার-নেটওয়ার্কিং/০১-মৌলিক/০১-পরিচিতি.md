# নেটওয়ার্কিং পরিচিতি (Introduction to Networking)

## 📖 নেটওয়ার্ক কী?

**নেটওয়ার্ক (Network)** হলো দুই বা ততোধিক কম্পিউটার বা ডিভাইসের এমন একটি সংযোগ যেখানে তারা একে অপরের সাথে ডেটা আদান-প্রদান করতে পারে।

### সহজ উদাহরণ:
- আপনার বাসার Wi-Fi এ ফোন, ল্যাপটপ সব কানেক্টেড - এটাই একটা নেটওয়ার্ক
- অফিসের সব কম্পিউটার একসাথে প্রিন্টার শেয়ার করছে - এটাও নেটওয়ার্ক
- ইন্টারনেট - পৃথিবীর সবচেয়ে বড় নেটওয়ার্ক

---

## 🎯 নেটওয়ার্কিং কেন শিখবেন?

1. **ক্যারিয়ার সুযোগ**: Network Engineer, System Admin, Security Analyst
2. **হাই স্যালারি**: IT সেক্টরে ভালো বেতনের চাকরি
3. **গ্লোবাল ডিমান্ড**: সব দেশে নেটওয়ার্ক প্রফেশনালদের চাহিদা
4. **সার্টিফিকেশন**: CCNA, CCNP, CompTIA Network+

## 🌐 নেটওয়ার্কের মূল উপাদান (Components)

### ১. নোড (Node)
নেটওয়ার্কে যুক্ত প্রতিটি ডিভাইসকে নোড বলে।
- কম্পিউটার
- প্রিন্টার
- স্মার্টফোন
- সার্ভার

### ২. মিডিয়া (Media)
ডেটা পাঠানোর মাধ্যম।
- **তারযুক্ত**: Ethernet Cable, Fiber Optic
- **তারবিহীন**: Wi-Fi, Bluetooth

### ৩. প্রোটোকল (Protocol)
যোগাযোগের নিয়মকানুন।
- TCP/IP
- HTTP
- FTP

### ৪. নেটওয়ার্ক ডিভাইস
- Router
- Switch
- Hub
- Modem

## 🔑 গুরুত্বপূর্ণ টার্ম (Key Terms)

### Bandwidth (ব্যান্ডউইথ)
- একটি নেটওয়ার্ক কতটুকু ডেটা বহন করতে পারে তার পরিমাপ
- সাধারণত Mbps বা Gbps এ মাপা হয়
- উদাহরণ: 100 Mbps মানে প্রতি সেকেন্ডে 100 Megabit ডেটা

### Latency (লেটেন্সি)
- ডেটা এক জায়গা থেকে অন্য জায়গায় যেতে যে সময় লাগে
- মিলিসেকেন্ড (ms) এ মাপা হয়
- কম latency = ভালো পারফরম্যান্স

### Throughput (থ্রুপুট)
- প্রকৃতপক্ষে কতটুকু ডেটা ট্রান্সফার হচ্ছে
- Bandwidth হলো সর্বোচ্চ ক্ষমতা, Throughput হলো আসল পারফরম্যান্স

### Packet (প্যাকেট)
- ডেটাকে ছোট ছোট অংশে ভাগ করে পাঠানো হয়
- প্রতিটি অংশকে প্যাকেট বলে

## 📐 নেটওয়ার্কের মাপকাঠি

```
বিট (bit) - সবচেয়ে ছোট একক
1 Byte = 8 bits
1 Kilobyte (KB) = 1024 Bytes
1 Megabyte (MB) = 1024 KB
1 Gigabyte (GB) = 1024 MB
1 Terabyte (TB) = 1024 GB
```

### স্পিড মাপা:
```
1 Kbps = 1000 bits per second
1 Mbps = 1000 Kbps
1 Gbps = 1000 Mbps
```
