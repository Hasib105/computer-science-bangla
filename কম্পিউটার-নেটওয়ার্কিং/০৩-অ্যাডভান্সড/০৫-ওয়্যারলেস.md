# ওয়্যারলেস নেটওয়ার্ক (Wireless Network)

## 📖 Wireless Network কী?

**Wireless Network** হলো এমন network যেখানে devices **radio waves** দিয়ে communicate করে, physical cable ছাড়াই।

```
Wired Network:              Wireless Network:
                            
┌────┐ ═══════ ┌────┐      ┌────┐  ～～～  ┌────┐
│ PC │  Cable  │ PC │      │ PC │  WiFi  │ PC │
└────┘ ═══════ └────┘      └────┘  ～～～  └────┘
```

## 📻 2.4 GHz vs 5 GHz

```
                2.4 GHz                    5 GHz
            ┌─────────────┐          ┌─────────────┐
            │ ～～～～～～～  │          │ ～～～～      │
            │   Longer    │          │   Shorter   │
            │   Range     │          │   Range     │
            │             │          │             │
            │   Slower    │          │   Faster    │
            │   Speed     │          │   Speed     │
            │             │          │             │
            │   More      │          │   Less      │
            │Interference │          │Interference │
            └─────────────┘          └─────────────┘
```

| Feature | 2.4 GHz | 5 GHz |
|---------|---------|-------|
| Range | দূর পর্যন্ত | কম range |
| Speed | ধীর | দ্রুত |
| Interference | বেশি (microwave, Bluetooth) | কম |
| Wall penetration | ভালো | খারাপ |
| Channels | 3 non-overlapping | 23+ |
| Best for | IoT, বড় ঘর | Streaming, gaming |

### AP vs Router vs Modem

| Device | Function |
|--------|----------|
| Modem | ISP signal convert করে |
| Router | Routing + NAT + DHCP |
| Access Point | শুধু wireless signal broadcast |
| Wireless Router | Router + AP combined |

Ch1    Ch2    Ch3    Ch4    Ch5    Ch6    Ch7    Ch8    Ch9   Ch10   Ch11
│──────│──────│──────│──────│──────│──────│──────│──────│──────│──────│

### Best Practices

```

✓ WiFi Analyzer app দিয়ে crowded channel দেখুন
✓ Least crowded channel বাছুন
```

## 🔐 WiFi Security

### Security Evolution

```
WEP (1997) → WPA (2003) → WPA2 (2004) → WPA3 (2018)
   ❌           △            ✓            ✓✓
 Broken      Weak        Standard      Strongest
```

### WEP (Wired Equivalent Privacy)

```
┌─────────────────────────────────────────────┐
│                  WEP ❌                       │
├─────────────────────────────────────────────┤
│ • 1997 সালের                                │
│ • RC4 encryption (weak)                     │
│ • Static key                                │
│ • মিনিটে crack করা যায়                      │
│ • কখনো ব্যবহার করবেন না!                    │
└─────────────────────────────────────────────┘
```

### WPA (WiFi Protected Access)

```
┌─────────────────────────────────────────────┐
│                  WPA △                       │
├─────────────────────────────────────────────┤
│ • TKIP (Temporal Key Integrity Protocol)    │
│ • Dynamic key                               │
│ • WEP এর চেয়ে ভালো কিন্তু এখন weak         │
│ • Avoid if possible                         │
└─────────────────────────────────────────────┘
```

### WPA2 (Current Standard)

```
┌─────────────────────────────────────────────┐
│                 WPA2 ✓                       │
├─────────────────────────────────────────────┤
│ • AES encryption (strong)                   │
│ • CCMP protocol                             │
│ • বর্তমান standard                          │
│                                             │
│ Two modes:                                  │
│ ├── WPA2-Personal (PSK): Home use           │
│ │   Pre-Shared Key (password)               │
│ │                                           │
│ └── WPA2-Enterprise: Corporate              │
│     RADIUS server authentication            │
└─────────────────────────────────────────────┘
```

### WPA3 (Latest)

```
┌─────────────────────────────────────────────┐
│                 WPA3 ✓✓                      │
├─────────────────────────────────────────────┤
│ New Features:                               │
│ • SAE (Simultaneous Authentication)         │
│   - Resistant to offline attacks            │
│ • 192-bit encryption (Enterprise)           │
│ • Forward secrecy                           │
│ • Easy Connect (QR code pairing)            │
│ • Protected management frames               │
└─────────────────────────────────────────────┘
```

## 🏢 Enterprise vs Personal

```
WPA2-Personal (Home):
┌────────┐   Same Password    ┌────────┐
│ User A │ ───────────────────│        │
└────────┘    "MyWiFi123"     │   AP   │
┌────────┐ ───────────────────│        │
│ User B │   Same Password    └────────┘
└────────┘


WPA2-Enterprise (Office):
┌────────┐   Username/Pass    ┌────────┐    ┌────────┐
│ User A │ ───────────────────│   AP   │────│ RADIUS │
└────────┘   user.a / pass1   │        │    │ Server │
┌────────┐ ───────────────────│        │────│        │
│ User B │   user.b / pass2   └────────┘    └────────┘
└────────┘   Different creds!

Enterprise advantages:
✓ Individual credentials
✓ User tracking
✓ Certificate-based auth option
✓ Revoke single user access
```

## 📡 SSID এবং Hidden Network

### SSID (Service Set Identifier)

```
SSID = WiFi network এর নাম

Available Networks:
┌──────────────────────────────┐
│ 📶 Home_WiFi_5G         🔒   │
│ 📶 Neighbor_Network     🔒   │
│ 📶 Office_Guest              │  ← Open (no lock)
│ 📶 CoffeeShop_Free          │
│ + Add hidden network...      │
└──────────────────────────────┘
```

### Hidden SSID

```
Regular AP:
AP broadcasts: "My name is HomeWiFi!"
Client: "I see HomeWiFi, connecting..."

Hidden SSID:
AP: (silent)
Client: "Anyone named HomeWiFi here?"
AP: "Yes, that's me!"

⚠️ Hidden SSID is NOT security!
   Tools can still find it!
```

## ⚠️ Wireless Threats

### ১. Evil Twin Attack

```
Legitimate AP:               Evil Twin:
SSID: "CoffeeShop"          SSID: "CoffeeShop"
┌──────────┐                ┌──────────┐
│  Real AP │                │ Hacker's │
│   📶     │                │    AP    │
└──────────┘                └─────┬────┘
      │                           │
      │     ┌──────────┐          │
      └────→│  Victim  │←─────────┘
             │   📱    │
             └──────────┘
       
Victim connects to fake AP!
Hacker sees all traffic!
```

### ২. Deauthentication Attack

```
Attacker sends fake "disconnect" packets

AP ←── Fake Deauth ─── Attacker
       packets
   ↓
Client disconnected!
May connect to Evil Twin
```

### ৩. WPS Attack

```
WPS PIN Attack:
┌─────────────────────────────────────────────┐
│ WPS PIN: 8 digits                           │
│ But splits into: 4 + 3 + 1 (checksum)       │
│                                             │
│ Bruteforce: 10^8 = 100 million tries        │
│ Reality:    10^4 + 10^3 = 11,000 tries!     │
│                                             │
│ ⚠️ Disable WPS on your router!              │
└─────────────────────────────────────────────┘
```

## 💻 Wireless Commands

### Windows

```cmd
# Available networks দেখুন
netsh wlan show networks

# Connected network details
netsh wlan show interfaces

# Saved networks list
netsh wlan show profiles

# Saved WiFi password দেখুন
netsh wlan show profile name="WiFiName" key=clear

# WiFi disconnect
netsh wlan disconnect

# Connect to network
netsh wlan connect name="WiFiName"
```

### Linux

```bash
# Scan for networks
sudo iwlist wlan0 scan

# Connection status
iwconfig wlan0

# Network Manager CLI
nmcli device wifi list
nmcli device wifi connect "SSID" password "PASSWORD"
```

## 📡 Advanced Wireless Concepts

### MIMO (Multiple Input Multiple Output)

```
Single Antenna:              MIMO (Multiple Antennas):
     │                        │ │ │   │ │ │
     │  One stream            │ │ │   │ │ │
     ▼                        ▼ ▼ ▼   ▼ ▼ ▼
  ┌─────┐                  ┌─────────────────┐
  │ AP  │                  │   4x4 MIMO AP   │
  └─────┘                  └─────────────────┘
                           Multiple streams = faster!
```

### MU-MIMO (Multi-User MIMO)

```
Regular MIMO:              MU-MIMO:
    ┌────┐                    ┌────┐
    │ AP │                    │ AP │
    └──┬─┘                    └──┬─┘
       │                         │ \ \ \
       ↓ One at a time           ↓  ↓  ↓  ↓
    ┌────┐                    ┌────┐┌────┐┌────┐
    │Dev1│                    │Dev1││Dev2││Dev3│
    └────┘                    └────┘└────┘└────┘
    then Dev2...              Simultaneous!
```

### Mesh WiFi

```
Traditional:                    Mesh Network:
                                
    ┌────┐                     ┌────┐───┌────┐
    │ AP │                     │Node│   │Node│
    └────┘                     └────┘   └──┬─┘
      📶                          📶    📶 │
   Limited                      ┌────┐←──┘
   coverage                     │Node│
                                └────┘ 📶
                                Whole home coverage!
```
